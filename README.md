MidiShip-M is a professional motorized MIDI controller with four 100 mm touch-sensitive faders, an encoder with button, OLED display, and built‑in Wi‑Fi. Designed for musicians, producers, and studio engineers, it brings tactile control to your DAW with two main protocols: MIDI CC (7‑bit or 14‑bit) and HUI emulation for Pro Tools, Logic Pro, Cubase, and others.
Key features:
4 motorized faders with capacitive touch sensors – automatic fader follow in CC mode, instant motor disengage on touch.
Encoder with button – navigate presets, adjust values, control transport, and access menus.
OLED 128×64 display – shows preset numbers, CC values, HUI channel states (mute/solo), transport, and more.
USB‑C MIDI class‑compliant – plug‑and‑play on macOS, Windows, Linux.
Built‑in Wi‑Fi & Web Interface – configure presets, motor settings, and system options from any browser; OTA firmware updates.
10 user presets – store CC mappings for faders and encoder, MIDI channel per preset.
Service Menu – fine‑tune motor parameters, reset presets, reset Wi‑Fi, enable/disable motors globally, control Identity Reply
Perfect for studio production, live performance, and DAW control.

Firmware Update:
1.  Open the web interface and click Firmware Update.
2. On the page that appears, select the downloaded firmware file (.bin extension).
3. The update process will start automatically.
4. Wait for the progress bar to complete. The controller will reboot with the new firmware.

⚠️ Important: Do not turn off the power or disconnect the USB cable while the update is in progress!

Обновление прошивки:
1. Зайдите на страницу управления, нажмите Firmware Update.
2. На открывшейся странице выберите скачанный файл прошивки с расширением .bin.
3. Начнётся автоматическая прошивка.
4. Дождитесь завершения (индикатор прогресса). Контроллер перезагрузится с новой прошивкой.
   
⚠️ Важно: не выключайте питание и не отключайте USB во время обновления!
